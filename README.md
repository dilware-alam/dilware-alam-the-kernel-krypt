# The Kernel Krypt


Dark-themed cybersecurity blog scaffold for GitHub Pages.


## Usage
1. Replace `assets/img/logo.png` with your logo.
2. Edit `_config.yml` to set `url` and `baseurl`.
3. Create posts in `_posts/` as Markdown files with Jekyll front matter.
4. Push to GitHub and enable Pages under repository Settings â†’ Pages.


## Create a post (example filename)
`_posts/2025-12-10-react2shell-malware-explained.md`


Front matter example:


```markdown
---
title: "React2Shell Malware Explained"
date: 2025-12-10
layout: post
categories: [malware, analysis]
---


Your post content here.
